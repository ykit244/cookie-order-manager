# 🍪 饼干订单管理系统 PWA

妈妈的手工饼干订单管理小助手 - 一个支持云端同步的渐进式网页应用

## ✨ 主要特点

- 🔐 **Google 账号登录** - 安全的身份认证，每个账号数据独立
- ☁️ **云端数据同步** - 使用 Firebase，多设备实时同步
- 📱 **PWA 支持** - 可安装到主屏幕，体验接近原生 app
- 🎯 **离线支持** - 即使断网也能继续工作
- 📊 **完整统计** - 营业额追踪、年度目标、产品销售分析

## 📱 核心功能

### 订单管理
- ✅ 完整的订单生命周期管理
  - 待处理 → 制作中 → 已完成 → 已交付
- ✅ 订单详细信息
  - 客户信息（姓名、电话、地址）
  - 产品明细（产品、大小、数量、单价）
  - 订单日期和交付日期
  - 订单备注
- ✅ 智能过滤
  - 按状态过滤
  - 按交付时间过滤（已过期/即将到来）
  - 搜索（客户姓名、电话、订单号）
- ✅ 订单保护
  - 转为已交付前需要确认
  - 只有待处理订单可以删除
  - 已交付订单不可编辑

### 客户管理
- ✅ 客户信息维护
  - 姓名、电话、地址、备注
- ✅ 快速选择客户
  - 新建订单时快速选择常客
  - 客户信息自动填充
- ✅ 智能同步
  - 修改客户信息自动更新所有相关订单

### 产品管理
- ✅ 灵活的产品配置
  - 支持多个大小选项（如：小、中、大）
  - 每个大小独立定价
  - 产品单位（盒/包/个）
  - 产品描述

### 日历视图
- ✅ 月度日历展示
- ✅ 交付日期可视化
- ✅ 每日订单概览
- ✅ 点击查看当日订单详情
- ✅ 客户名字过长自动裁剪

### 统计分析
- ✅ 主页统计面板
  - 年度营业额（仅计算已交付订单）
  - 营业额目标和进度条
  - 各状态订单数量统计
- ✅ 已交付产品统计
  - 各产品销量
  - 各产品营业额
- ✅ 年份管理
  - 支持多年份数据
  - 快速切换年份
  - 创建新年份

### 数据安全
- ✅ Google OAuth 认证
- ✅ 每个账号数据完全隔离
- ✅ Firebase 云端安全存储
- ✅ 自动备份，不会丢失
- ✅ 多设备实时同步

## 🚀 快速开始

### 在线使用（推荐）

应用已部署到 Firebase：
```
https://cookie-order-management.web.app/
```

直接访问即可使用，无需安装任何软件！

### 📱 安装到手机主屏幕

#### iOS (Safari)
1. 在 Safari 中打开网址
2. 点击底部的 **分享** 按钮（方框带向上箭头）
3. 向下滚动，选择 **"添加到主屏幕"**
4. 输入名称（例如："饼干订单"）
5. 点击 **"添加"**

#### Android (Chrome)
1. 在 Chrome 中打开网址
2. 点击右上角 **菜单**（三个点）
3. 选择 **"添加到主屏幕"** 或 **"安装应用"**
4. 点击 **"添加"** 或 **"安装"**

完成后，手机桌面会出现应用图标，点击后全屏打开，体验接近原生 app！
